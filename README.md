# 🤖 Clara - Study Chatbot 💫

## 📌 Description
Clara is a simple rule-based study chatbot built using Python and Streamlit. It helps students with basic academic questions and general conversations.

## ⚙️ Technologies Used
- Python
- Streamlit

## 🧠 Approach
This chatbot uses a rule-based approach with predefined responses to ensure fast and accurate replies.

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run chatbot.py

3. Open in browser:
   http://localhost:8501

## ✨ Features
- Chat-style interface
- Instant responses
- Study-related assistance
- Beginner-friendly design

## ⚠️ Challenges
- Handling different user inputs
- Making responses feel natural

## ✅ Solution
- Used keyword matching
- Added friendly fallback responses
